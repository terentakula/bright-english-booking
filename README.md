# Bright English Booking

Bright English Booking — demo-приложение для онлайн-школы английского языка.
Проект показывает полный сценарий записи на занятие: от просмотра курсов до управления заявками в админ-панели.

## Стек

React
TypeScript
Vite
Tailwind CSS
Zustand
React Router
Motion / Framer Motion

## Основной функционал

### Для гостя

Просмотр главной страницы
Просмотр списка курсов
Отправка заявки на занятие
Переход к демо-входу

### Для ученика

Вход в демо-кабинет ученика
Просмотр своих заявок
Отображение статуса заявки
Плавное появление карточек заявок

### Для администратора

Просмотр всех заявок
Фильтрация и сортировка заявок
Изменение статуса заявки
Просмотр учеников и истории записей
Создание новых курсов
Редактирование существующих курсов
Удаление и отключение курсов
Обновлённый курс автоматически перемещается в начало списка

## Демо-роли

В проекте реализованы три роли:

Guest — обычный посетитель сайта
Client — ученик
Admin — администратор

Авторизация сделана в demo-формате через Zustand и localStorage.
Проект не использует реальный backend, так как цель — показать UI, бизнес-логику и клиентский сценарий.

## Особенности проекта

Адаптивная вёрстка
Desktop и mobile navigation
Боковое меню на небольших экранах
Protected routes по ролям
CRUD для курсов
Работа со статусами заявок
Плавные page transitions
Анимации появления карточек
Мягкие hover-эффекты
Единая цветовая система для статусов
LocalStorage persistence через Zustand
